# Plague Simulator

Learning project which aims to simulate the evolution of a disease inside 
a defined population using Multi Agent system in Unity.

## Features

| Feature                    | Progress                                                       |
|-----------------------     |----------------------------------------------------------------|
| Citizen Agent              | Done                                                           |
| Mayor Agent                | Done                                                           |
| Environment                | Done                                                           |
| Use of real disease models | In dev                                                         |
